Welcome to Polish Memes, the ultimate bot that will immerse you in the world of Polish memes.

Polish Memes Bot guarantees 100% security. It does not store any detailed data about clients or their servers. It operates 24/7 and is fully safe for work (SFW).

Terms of Service / Privacy Policy

The bot was originally designed in JavaScript but was later rewritten in Python and now operates using the Discord.js library.

Polish Memes Bot is hosted on Daki.cc hosting and can handle a large number of servers despite being free.

Getting Started

Using Polish Memes Bot is easy! Simply invite the bot to your server of choice and use the command "/meme" to generate a cool meme for yourself. Enjoy!

Commands

  • Meme
    • Description: Generate a cool meme for yourself.
    Usage: /meme
    Example: /meme
    Note: This command will generate a funny meme for you sourced from Reddit.
  • Ping
    • Description: Check the bot's response time.
    Usage: /ping
    Example: /ping
    Note: This command will measure the bot's latency and response time.
  • Info
    • Description: Display information about the bot.
    Usage: /info
    Example: /info
    Note: This command will provide detailed and important information about the bot.
  • Help
    • Description: Get help and information about available commands.
    Usage: /help
    Example: /help
    Note: This command will display a list of available commands and their usage.
  • Profile
    • Description: View user profile.
    Usage: /profile [@user]
    Example: /profile @john
    Note: This command will display the profile of the specified user or your own profile if no user is mentioned.
  • Reputation
    • Description: Give reputation points to a user.
    Usage: /reputation [@user]
    Example: /reputation @jane
    Note: This command will give reputation points to the specified user.
  • Top
    • Description: View leaderboard for levels, balances, or reputation.
    Usage: /top [level/balance/reputation]
    Example: /top level
    Note: This command will display the top users based on the specified category (level, balance, or reputation).
  • Daily
    • Description: Claim daily reward.
    Usage: /daily
    Example: /daily
    Note: This command will allow you to claim a daily reward.
  • Transfer
    • Description: Transfer currency to another user.
    Usage: /transfer [@user] [amount]
    Example: /transfer @john 100
    Note: This command will transfer the specified amount of currency to the mentioned user.
  • Bet
    • Description: Bet currency for a chance to win more.
    Usage: /bet [amount]
    Example: /bet 50
    Note: This command will allow you to bet the specified amount of currency.
  • Wallet
    • Description: View user's wallet balance.
    Usage: /wallet [@user]
    Example: /wallet @john
    Note: This command will display the wallet balance of the specified user or your own balance if no user is mentioned.
