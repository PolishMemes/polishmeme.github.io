Introduction

A bot that will draw you into Polish memes.

About this Bot

The bot guarantees 100% security. It does not store any detailed data about clients or their servers. It operates 24/7 and is fully SFW.

Technology

The bot was designed in java script, but was later rewritten in python and now operates in discord.js.

The bot is hosted on https://daki.cc hosting and copes admirably with the number of servers it runs on despite being free.

Getting Started

The bot is not complicated to use. All you have to do is invite him to the server of your choice, give the command: /meme and that's it. :D

In-bot help

If you encounter any problems with the bot, please contact us on the discord https://discord.com/invite/ShXq5Q2kQQ.
If you have any questions, you are also welcome!

Adding the bot

To add the bot to your server click here.

The bot does not have too many requirements. So you don't need to give it an administrator or server management.

Commands

Syntax:
  • < > : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

Meme Generator

  • Meme
    • Description : Generate a cool meme for yourself. ;D
    Usage : /meme
    Example : /meme
    Note : Should draw some funny picture for you from Reddit.
  • Ping
    • Description : Ping-Pong.
    Usage : /ping
    Example : /ping
    Note : Checks the bot's response time.
  • Servers
    • Description : You can see how many servers the bot is on.
    Usage : /servers
    Example : /servers
    Note : Displays the number of servers the bot is on.
  • Bot
    • Description : Information about the bot
    Usage : /bot
    Example : /bot
    Note : Little detailed/important information about the bot is displayed.
