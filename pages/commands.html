Introduction

A bot that will draw you into Polish memes.

About this Bot

The bot guarantees 100% security. It does not store any detailed data about clients or their servers. It operates 24/7 and is fully SFW.

Technology

The bot was designed in java script, but was later rewritten in python and now operates in discord.js.

The bot is hosted on https://daki.cc hosting and copes admirably with the number of servers it runs on despite being free.

Getting Started

The bot is not complicated to use. All you have to do is invite him to the server of your choice, give the command: /meme and that's it. :D

In-bot help

There is also a in-bot gettingstarted command that you can use if you know the installation process.
You also have the option to get help on all the bot commands right through discord with the use of the bot's internal help command.

Important: You must agree to our imagepolicy and matter regarding it will be fairly strict.
Use dh imagepolicy for more information.

Adding the bot

To add the bot to your server click here.

The bot does not have too many requirements. So you don't need to give it an administrator or server management.

Commands

Syntax:
  • < > : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

Meme Generator

  • Anime
    • Description : Generate a cool meme for yourself. ;D
    Usage : /meme
    Example : /meme
    Note : Should draw some funny picture for you from Reddit.
  • Ping
    • Description : Let's you search any manga with the help of Anilist API.
    Usage : dh manga < your manga >
    Example : dh manga death note
    Note : The manga name given can be case in-sensetive.
  • Doujin
    • Description : Let's you search and read doujins from nhentai.net.
    Usage : dh doujin < your code >
    Example : dh doujin 000000
    Aliases: search, id
    Note : By default doujins with tags which are against Discord TOS are not available through this command.
  • Character
    • Description : Let's you search information about the characters listed in Anilist.
    Usage : dh character < character name >
    Example : dh character light yagami
    Note : All the characters are not availabe only those ones at Anilist.co. So, if you want you can add it there.
  • Sauce (Beta)
    • Description : Let's you search the source of the post image.
    Usage : dh sauce < link >
    Aliases : imagesearch
